# Running this       

Download the repository folder as it is.   
In the command prompt opened from the backend folder, run these commands      
npm install            
npm install express mongoose cors dotenv                
npm install -D nodemon        
node seed.js                
npm run dev                    

backend: http://localhost:4000            

Again in the command prompt opened from the frontend folder, run these commands                       
npm install                 
npm audit fix --force  {If there are any issues}         
npm install react-router-dom         
npm install react-router-dom@6        
npm start        

frontend: http://localhost:3000
